# Everything downloader Bot

This Telegram bot can download almost anything :)

[koyu.space](http://koyu.space)


### Used Libraries/Applications

- NodeJS (latest version)
- FFMPEG
- LAME
- eyeD3
- node-acoustid
- chromaprint
- Python 2
- Python 3
- youtube-dl
- wget
- Sacad
